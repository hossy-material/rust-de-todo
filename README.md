# Learning Rust

**Webアプリ開発で学ぶRust言語入門**を読んでRustのお勉強

### フロントエンド
- React
- MUI

### バックエンド
- Rust
- axum
- PostgreSQL
- sqlx

## 起動
- DB
```sh
cd backend
make db
```

- サーバー
```sh
cd backend
make dev
```

- フロント
```sh
cd frontend
npm install
npm run dev
```
